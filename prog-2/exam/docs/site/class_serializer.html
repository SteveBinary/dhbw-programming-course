template<class T >
vector< T > Serializer< T >::deserialize ( ) const

Reads a file and creates objects according to the type parameter. The given type must extend Serializable. Each line in the input file correspondes to one dataset which is used for object-initialization. Each line has to have the correct format for deserialization. The format is given by the type parameter. The created objects are collected into a vector.

Returns
The vector of all succesfully created objects.
