◆ extendToLength()

std::string Util::extendToLength ( std::string const &  s,
unsigned int const  targetLength 
)

Adds white spaces to the given string until the target length is reached. If the target length is smaller than the initial length of s, the result is the same as s. The number of added white spaces is targetLength - initial length of s.

Parameters
s- The string to extend with white spaces. (UN-MODIFIED)
targetLength- The length the string should have after adding the white spaces.
Returns
The extended string.
Caution
The length of the string is the number of bytes! Characters like 'รถ' could lead to unexpected results!
