Person-Sorting-Machine
person.deserializeFromString

Sets the person's attributes according to the given string. The string has to be in the following format: "firstname,lastname,age,salary"

("John,Smith,42,2899.99");

Parameters
s- The string used to set the person's attributes.
Exceptions
DeserializationException- When the string cannot be used to set the person's attributes because of a wrong format.
